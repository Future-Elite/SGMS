## 项目背景

随着人工智能和计算机视觉技术的发展，手势识别作为自然交互方式，正逐步渗透到智能家居、医疗康复、教育培训、娱乐等领域。当前市场对非接触式交互需求激增，尤其在5G网络普及下，实时手势控制成为提升用户体验的核心技术方向。

本项目旨在开发一款基于电脑端的低成本手势识别系统，通过普通摄像头实现手势采集与识别，解决传统交互方式（如触控屏、鼠标）在特定场景（如湿手操作、沉浸式交互）中的局限性。
